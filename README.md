AingDesk-Next是在AingDesk基础上进行的前后端分离版本，它是一款简单好用的AI助手，支持知识库、模型API、MCP、联网搜索、智能体。 它是一个不错的基座模型，也可以用来快速切入AI项目，例如AI本地化数据取证、律师助手等等本地化AI项目。 在此感谢AingDesk项目的开源，访问链接https://github.com/aingdesk/AingDesk。 如果觉得不错，欢迎给一个star星星，支持我们不断更新变得更好。
--- 
aingdesk_api项目是python开发的后台接口用于业务逻辑实现，frontend项目是electron的前端项目用于展示界面
